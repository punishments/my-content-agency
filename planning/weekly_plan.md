# Weekly Execution Plan

Six posts per week. Four Attract, one Nurture, one Position. Three Talking, three Non-talking. Two High intensity, four Low intensity.

---

## Weekly Schedule

| Day | Activity |
|-----|---------|
| **Monday** | Run `docs/weekly_idea_session.md` — triage, balance, open proposals |
| **Monday–Tuesday** | Run pipeline Steps 1–5 for each post (proposal stage) |
| **Tuesday** | Review and approve proposals via `planning/weekly_signoff.md` |
| **Wednesday** | Copywriter writes approved copy |
| **Wednesday–Thursday** | QA Editor reviews all copy |
| **Wednesday** | Non-talking post published |
| **Thursday–Friday** | Film talking content |
| **Friday** | Talking post published |
| **Weekend** | Analytics review — update `/analytics/raw/` stubs |

The sequence is fixed. The specific days are flexible.
