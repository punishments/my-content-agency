# Weekly Control

Single source for weekly approval and execution tracking.

Cadence targets: `docs/execution_system.md` (canonical)

> Copy the week template below each Monday. Most recent at top.

---

## How to Use

1. **Monday** — run `docs/weekly_idea_session.md`, check `planning/content_backlog.md`, confirm 6-slot distribution before opening proposals
2. **Proposal stage** — Concept agent populates each slot when READY FOR APPROVAL
3. **Approval** — review each slot, mark decision, no scripting begins before decision is logged
4. **Execution** — track script, QA, published status per slot as the week progresses

---

## Status Key

| Symbol | Meaning |
|--------|---------|
| `[ ]` | Not started |
| `[~]` | In progress |
| `[x]` | Done |
| `[✓]` | Published |
