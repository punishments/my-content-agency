# Proposals

This folder holds all content proposals from Step 5 (Creative Director recommendation) through Step 6 (user approval).

---

## What Lives Here

- All proposals in status: DRAFT, READY FOR APPROVAL, REVISION REQUESTED
- Approved proposals (stay here after approval — the script moves to `/scripts/`)
- Rejected proposals (stay here, marked REJECTED, for reference)

## What Does NOT Live Here

- Final scripts → those save to `/scripts/[YYYY-MM-DD]_[slug].md`
- Analytics stubs → those save to `/analytics/raw/[YYYY-MM-DD]_[slug].md`

---

## File Naming

```
YYYY-MM-DD_[slug]_proposal.md
```

Examples:
```
2026-03-31_overthinking-trust-problem_proposal.md
2026-03-31_girl-in-her-30s-behind_proposal.md
```

Use the publication date (or target date), not the creation date.

---

## Proposal Lifecycle

```
Created (Step 5 complete) → READY FOR APPROVAL
        ↓
User reviews in planning/weekly_control.md
        ↓
APPROVED → Copywriter picks up → script saved to /scripts/
REVISION REQUESTED → Returns to specified step → re-submitted here
REJECTED → Stays here, status updated to REJECTED
```

---

## Maintenance

- Clear approved and rejected proposals at the end of each month
- If a proposal is approved but the script hasn't been written yet, the proposal file stays here until the script is filed
- Do not delete REJECTED proposals immediately — they are useful reference for what was tried and why it didn't proceed
