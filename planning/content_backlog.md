# Content Backlog

Ideas that have passed triage but are not scheduled for this week. Drawn from during the Monday idea session when slots are available.

---

## How to Use

- **Add ideas here** after they pass the triage filter in `docs/content_decision_system.md` but can't be scheduled this week
- **Pull from here first** at the start of each Monday session before generating new ideas
- **Update priority** as the account stage or strategy evolves
- **Remove ideas** that are no longer relevant — stale ideas produce stale content

---

## Priority Levels

| Priority | Meaning |
|----------|---------|
| **High** | Ready to schedule. Fully triaged. Pull into next available week. |
| **Medium** | Good idea but needs a stronger angle or more material before executing. |
| **Low** | Interesting but not urgent. Keep for reference. |
| **Hold** | Time-sensitive — do not schedule until a specific trigger or date. |

---

## Backlog Template

Copy this block for each idea:

```
---
**Idea:** [One sentence — the raw idea]
**Source:** [Experience / Learning / Struggle / Research / 7x7 insight]
**Connects to:** [Pillar — Skill / Mindset / Lifestyle / Passions] + [belief or audience pain]
**Likely mission:** [Attract / Nurture / Position]
**Likely format:** [Talking / Non-talking]
**Priority:** [High / Medium / Low / Hold]
**Added:** [YYYY-MM-DD]
**Notes:** [Anything that will help when this gets scheduled — angle seeds, execution notes, what's missing]
---
```
