# Recent Working Set

Short-term iteration context. Last 3–5 scripts in development or recent cycles.

Used for: Quick comparison, pattern identification within same week, style continuity.

**Optional.** Use if you need short-term reference. Not required for system function.

---

## Structure

```
recent/
└── [YYYY-MM-DD]_[slug]_final.md
└── [YYYY-MM-DD]_[slug]_final.md
└── ... (last 3–5 scripts only)
```

---

## When to Use

- Checking consistency within a week's output
- Quick reference to recent hook patterns
- Auditing tone drift week-to-week
- Short-term iteration context

---

## When NOT to Use

- As a library of all past work (use archive + analytics)
- For pattern extraction (use patterns/ directory)
- For long-term trend analysis

---

## Maintenance

- Keep only 3–5 most recent completed scripts
- Remove when a script ages >2 weeks
- Move aged scripts to `/archive/scripts/`

**Keep this lightweight.** Size limit: ~5 files max.
