# Content Prompt Frameworks (Structured Hooks)

This file defines structured hook frameworks used for content generation.

These frameworks provide:
- clarity
- repeatability
- fast execution

They MUST be used alongside:
- 7x7 research (what works)
- storytelling principles (problem → pursue → payoff)

---

# CORE PRINCIPLE

Frameworks = structure  
NOT performance.

Performance comes from:
- emotional specificity
- real language
- strong packaging

---

# HOW TO USE

When generating hooks:

1. Select a framework
2. Adapt it to:
   - a real internal thought
   - a real emotional struggle
3. Make it specific and relatable
4. Ensure it works in the first 3 seconds

---

# HOOK FRAMEWORKS

---

## 1. Pain Point Hook

### Template
- "If you’ve ever [pain], this is for you"
- "You’re not stuck — you’re just [misunderstood problem]"
- "I thought I was [fear], but I was actually [reframe]"

### Use When
- targeting emotional struggles
- building relatability

### Funnel
- Attract / Nurture

### Notes
- Must feel real, not generic
- Works best with internal thoughts

---

## 2. Audience Callout Hook

### Template
- "If you’re [specific audience], listen to this"
- "This is for people who [behavior/identity]"
- "If you're the type of person who [pattern], this will hit"

### Use When
- targeting a niche identity
- increasing relevance quickly

### Funnel
- Attract

### Notes
- Be specific (avoid broad labels like “entrepreneurs”)

---

## 3. Question Hook

### Template
- "Why do [group] always [problem]?"
- "Have you ever felt like [situation]?"
- "What if the reason you [problem] is actually [reframe]?"

### Use When
- creating curiosity
- prompting reflection

### Funnel
- Attract

### Notes
- Avoid vague or obvious questions

---

## 4. Contrarian Hook

### Template
- "Stop doing [common advice]"
- "This is why [common belief] is wrong"
- "Everyone says [X], but that’s exactly why you’re stuck"

### Use When
- challenging assumptions
- creating tension

### Funnel
- Position / Attract

### Notes
- Must be backed by a clear explanation

---

## 5. Statement / Reframe Hook

### Template
- "You’re not [problem], you’re [reframe]"
- "[Group] don’t struggle with [X], they struggle with [Y]"
- "You don’t need [thing], you need [truth]"

### Use When
- reframing identity
- delivering insight

### Funnel
- Position

### Notes
- Works well in talking formats

---

## 6. Instruction Hook

### Template
- "Do this before you [goal]"
- "Here’s what to do if [problem]"
- "Start doing this if you want [result]"

### Use When
- giving actionable advice
- clear value upfront

### Funnel
- Position / Convert

### Notes
- Must feel specific, not generic advice

---

## 7. Curiosity / Open Loop Hook

### Template
- "This is the moment before [event]"
- "No one talks about this, but…"
- "I didn’t realize this until [moment]"

### Use When
- increasing retention
- creating intrigue

### Funnel
- Attract

### Notes
- Must resolve later in the video

---

## 8. Internal Thought Hook (HIGH PRIORITY)

### Template
- "I thought I was [fear]"
- "I kept telling myself [limiting belief]"
- "I felt like I was [struggle]"
- "I didn’t realize I was [pattern]"

### Use When
- creating self-recognition
- emotional connection

### Funnel
- Attract / Nurture

### Notes
- This is your PRIMARY hook type
- Must feel like real inner dialogue
- Avoid polished phrasing

---

## 9. POV Hook

### Template
- "POV: you’re [specific situation]"
- "POV: you're the only one who [pattern]"
- "POV: you finally realize [truth]"

### Use When
- immersive storytelling
- relatable scenarios

### Funnel
- Attract

### Notes
- Works best with non-talking formats

---

## 10. Moral Framing Hook

### Template
- "[Person] said this in public…"
- "This is what they don’t want you to see"
- "This is actually insane"

### Use When
- triggering emotion (outrage, disbelief)
- high engagement

### Funnel
- Attract

### Notes
- Use carefully
- best for series content

---

# FORMAT ALIGNMENT RULES

---

## NON-TALKING (Silent Film)

Hook must:
- be readable instantly
- be emotionally clear
- be short (1–2 lines)

Avoid:
- complex sentences
- abstract phrasing

---

## TALKING (Voiceover / Talking Head)

Hook must:
- be spoken clearly in first sentence
- be emotionally direct
- avoid long buildup

---

# SYSTEM RULES

- Always include at least one framework hook
- Always pair with adaptive hooks (from 7x7)
- Prioritize emotional clarity over cleverness
- Optimize for first 3 seconds

---

# FINAL CHECK (BEFORE USING A HOOK)

Ask:

1. Would someone understand this instantly?
2. Does this feel like a real thought?
3. Would someone say “this is me”?
4. Does this fit the format?

If not → rewrite
