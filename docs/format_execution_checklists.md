# Format Execution Checklists

## Core Principle

Every piece of content must earn attention in the first 3 seconds and deliver a clear payoff by the end. These checklists are the quality gate — not a suggestion. If the output doesn't pass, revise before finalizing.

---

## Talking Content Checklist

Use this for any content where the creator speaks on camera.

| # | Standard | Pass? |
|---|----------|-------|
| 1 | **Hook (first 3 seconds):** Has a spoken hook, a visual hook, and a title hook | |
| 2 | **Narrative flow:** Clear story progression — no tangents, no confusion | |
| 3 | **Vocal delivery:** Language is simplified; uses inflection changes, strategic pauses, or repetition for emphasis | |
| 4 | **Overlays:** B-roll and text add to the video — not distracting; captions are included | |
| 5 | **Framing:** Well-lit; no app icons or visual clutter affecting visibility | |
| 6 | **Duration:** Falls within 35–90 seconds | |

---

## Non-Talking Content Checklist

Use this for any content with no on-camera speaking (text-on-screen, b-roll montage, etc.).

| # | Standard | Pass? |
|---|----------|-------|
| 1 | **Hook (first 3 seconds):** Has a visual hook and a title hook | |
| 2 | **Story arc:** Clear progression — no tangents, no confusion | |
| 3 | **Visual anchor:** Uses at least one of the following: Time (years, months, age) / Contrast (before vs. after) / Process (steps 1 → 2 → 3) | |
| 4 | **Readable text:** Each line can be read aloud and stands apart from the background | |
| 5 | **Anticipatory music:** Music builds — rising action insinuates a payoff | |
| 6 | **Duration:** Falls within 8–45 seconds | |

---

## How Claude Should Use This

- If a content idea is assigned **Talking**, evaluate it against the Talking checklist before returning output
- If a content idea is assigned **Non-Talking**, evaluate it against the Non-Talking checklist before returning output
- If the content **fails any item**, revise the hook, structure, or packaging — do not return a failing draft
- Use this as a **QA gate**, not a reference — the checklist runs before every finalized script or concept
