# Content Quality Checklist

Use AFTER script generation, BEFORE publishing.

---

## Core Principle

Content = Message + Format. Both must be correct.

---

## Universal Checks — All Content

**Hook**
- [ ] Clear in the first 3 seconds
- [ ] Creates immediate self-recognition
- [ ] A cold viewer can understand it instantly

**Story**
- [ ] Problem is clear
- [ ] Follows Problem → Pursue → Payoff
- [ ] No tangents or confusion

**Packaging**
- [ ] Someone with zero context can understand this
- [ ] Message is simple and direct

---

## Non-Talking Checklist (Silent Film)

**Hook**
- [ ] Visual + title hook appear immediately
- [ ] Hook is readable within 1–2 seconds

**Story Arc**
- [ ] Clear progression
- [ ] No confusion or jumps

**Visual Anchors** — must include at least one
- [ ] Time (years, age, timeline)
- [ ] Contrast (before vs. after)
- [ ] Process (step-by-step)

**Text Quality**
- [ ] Each line can be read quickly
- [ ] Each line stands alone
- [ ] No dense paragraphs

**Music**
- [ ] Builds anticipation
- [ ] Supports emotional payoff

**Duration**
- [ ] Target: 8–45 seconds
- [ ] No unnecessary length

---

## Talking Checklist (Voiceover / Talking Head)

**Hook**
- [ ] Spoken hook lands in the first sentence
- [ ] Visual + title support the hook

**Narrative Flow**
- [ ] Clear progression
- [ ] No rambling or over-explaining

**Vocal Delivery**
- [ ] Clear, simple language
- [ ] Inflection changes
- [ ] Strategic pauses or repetition used

**Overlays**
- [ ] Add clarity, not distraction
- [ ] Captions included

**Framing**
- [ ] Clean background
- [ ] Good lighting
- [ ] No UI distractions

**Duration**
- [ ] Target: 35–90 seconds

---

## Final Pass — Required

Before posting, ask:

1. Would someone say "this is me" in the first 3 seconds?
2. Is the story instantly clear?
3. Does the format feel clean and easy to follow?

**Any answer = NO → revise before posting.**
